## Instructions

### Compilation

The project uses a `Makefile` to compile the source files.

* **Compile the project:**
    `make`
    -> This generates the `fdf` executable.

* **Clean object files:**
    `make clean`

* **Full clean (objects + executables):**
    `make fclean`

* **Clean and compile :**
    `make re`

### Execution

To run the program:
`./fdf [file]`
